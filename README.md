# raylib
C + Raylib simple projects (to learn C code)

How to use (compiling and running):

tested on ArchLinux:

install raylib (from extra repo) :
# pacman -S raylib

and raygui (from AUR);
# https://aur.archlinux.org/packages/raygui

to compile source code: 
# gcc -Wall -Werror file.c -o executable_name -lraylib -lm

Enjoy!
